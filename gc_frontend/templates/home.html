Gestión de citas médicas

Bienvenido al sistema de gestión de un consultorio médico. Esta aplicación está diseñada para optimizar y facilitar las actividades relacionadas con la atención médica, permitiendo:

  • El registro seguro de usuarios con sus credenciales únicas.
  • La generación de turnos para citas médicas de manera eficiente.
  • El almacenamiento del historial médico actualizado después de cada consulta.
  • La creación de solicitudes de exámenes médicos por parte del médico tratante.
  • La emisión de recetas médicas basadas en diagnósticos realizados.
  • La visualización detallada de todos los datos asociados a los pacientes.

Explora las funcionalidades del sistema a través del menú de navegación para realizar tus tareas de manera rápida y eficiente.
